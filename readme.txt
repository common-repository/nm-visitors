This plugin is useful to check the page visits and daily visitors.

== Description ==

Google provides lots of basic tools to get the visitos but this plugin provides you data of all visitos, opened page their IP.

== Installation ==

1. Upload 'nm-redirect' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the NM Visitor
